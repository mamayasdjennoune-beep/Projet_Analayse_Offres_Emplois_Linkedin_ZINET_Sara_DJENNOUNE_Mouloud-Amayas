
# Projet : Analyse des Offres d'Emploi LinkedIn avec Snowflake

## Description du projet

Ce projet a pour objectif d’analyser le marché de l’emploi à partir de données issues de LinkedIn en mettant en œuvre une architecture **Medallion (Bronze, Silver, Gold)** sur **Snowflake**, complétée par une **application de visualisation interactive développée avec Streamlit**.



## Contenu du dépôt

- **README.md** : documentation générale du projet  
- **rapport_projet.md** : rapport détaillé du projet (architecture, traitements, analyses et résultats)  
- **linkedin_pipeline.sql** : script SQL complet comprenant les couches Bronze, Silver et Gold  
- **linkedin_streamlit_application.py** : application Streamlit de visualisation et d’analyse  
- **images/** : captures d’écran utilisées dans le rapport (résultats SQL et application Streamlit)
